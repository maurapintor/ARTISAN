# Lab session ARTISAN

## Getting started locally

```bash
conda create -n artisan-secml python=3.10
conda activate artisan-secml
conda install pip
pip install -r requirements.txt
```

Then you can launch the notebooks:

```bash
jupyter-notebook 01_intro_secml.ipynb
```
